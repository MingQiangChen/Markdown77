# 产品需求与 MVP

## 目标用户

Markdown77 面向需要长期沉淀知识的人：

- 学生
- 研究人员
- 程序员
- 内容创作者
- 产品经理
- 个人知识管理用户

## MVP 必须有的功能

### Markdown 编辑

- 支持标准 Markdown。
- 支持实时预览。
- 支持分屏编辑和预览。
- 支持快捷键。
- 支持图片粘贴。
- 支持代码块高亮。

### Vault 文件库

Vault 是一个普通文件夹。

```text
MyVault/
  日记/
    2026-06-03.md
  项目/
    Markdown77.md
  attachments/
    image.png
```

应用不应该把笔记锁进私有数据库。数据库只做索引缓存。

### 文件管理

- 文件树
- 新建笔记
- 重命名
- 删除
- 移动
- 最近打开
- 收藏

### 知识链接

- 支持 `[[双向链接]]`
- 支持 `#标签`
- 支持反向链接列表
- 支持未创建笔记提示

### 搜索

- 文件名搜索
- 全文搜索
- 标签搜索
- 链接搜索

## MVP 不建议一开始做

- 插件市场
- 复杂图谱视图
- 团队协作
- 实时多人编辑
- 自研云同步
- AI Agent
- 复杂主题系统

这些功能会明显增加复杂度，应该在核心体验稳定后再做。
