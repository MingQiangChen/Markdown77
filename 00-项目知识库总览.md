# Markdown77 项目知识库总览

Markdown77 的目标是做一个类似 Obsidian 的全平台 Markdown 编辑器，支持 iOS、Android、HarmonyOS NEXT、Windows、macOS、Linux。

## 产品定位

Markdown77 应该优先成为一个“本地优先”的知识管理工具，而不是只依赖云端数据库的在线编辑器。

核心原则：

- 用户数据是普通 Markdown 文件。
- 应用可以离线完整使用。
- 同步是增强能力，不是使用前提。
- 不锁死用户数据，用户可以随时用其他 Markdown 工具打开。
- 先做稳定的编辑、搜索、链接和同步，再做插件市场和 AI 功能。

## 建议开发阶段

第一阶段：桌面 MVP

- Windows、macOS、Linux
- Markdown 编辑与预览
- 文件树
- 本地 Vault
- 全文搜索
- 双向链接

第二阶段：移动端

- iOS
- Android
- 移动端文件访问
- 基础同步
- 图片和附件管理

第三阶段：HarmonyOS NEXT

- ArkTS + ArkUI 原生适配
- 文件权限适配
- 同步能力适配

第四阶段：高级功能

- 插件系统
- 图谱视图
- WebDAV / iCloud / OneDrive / 自研云同步
- 端到端加密
- AI 辅助写作
