# 图表、媒体与手写标注使用手册

本文说明 Markdown77 桌面 MVP 中图表、插图、多媒体链接、平板手写绘图标注的使用方式。

## 1. 图表

点击顶部工具栏的“图表”按钮，会向当前笔记插入一个图表代码块。

格式如下：

````markdown
```chart
title: 示例图表
计划: 40
进行中: 25
完成: 60
```
````

预览区会把 `chart` 代码块渲染成横向柱状图。

当前版本支持：

- `title` 作为图表标题。
- 每一行 `名称: 数字` 作为一个数据项。
- 自动按最大值计算柱状条长度。

当前版本暂不支持：

- 折线图。
- 饼图。
- 多系列图表。
- 从 CSV 或表格自动生成图表。

## 2. 插图与图片

点击顶部工具栏的“媒体”按钮，输入图片路径或 URL，会插入标准 Markdown 图片语法：

```markdown
![媒体](assets/example.png)
```

支持本地相对路径，也支持网络图片链接。

建议在 Vault 内建立附件目录：

```text
attachments/
  image.png
  demo.mp4
  voice.m4a
```

然后在笔记中引用：

```markdown
![项目截图](attachments/image.png)
```

## 3. 音频与视频

“媒体”按钮同样可以插入音频或视频链接。

视频示例：

```markdown
![演示视频](attachments/demo.mp4)
```

音频示例：

```markdown
![会议录音](attachments/meeting.m4a)
```

预览区会根据扩展名自动渲染为播放器。

当前支持的常见格式：

- 视频：`mp4`、`webm`、`mov`、`ogg`
- 音频：`mp3`、`wav`、`m4a`、`aac`、`flac`

## 4. 手写绘图标注

点击顶部工具栏的“绘图”按钮，会打开绘图面板。

可用输入方式：

- 鼠标
- 触控板
- 平板触控
- 手写笔

按钮说明：

- `清空`：清除当前画布。
- `插入`：把当前绘图保存成 PNG 图片并插入当前笔记。
- `X`：关闭绘图面板。

当前桌面版会把绘图结果保存到 Vault 内的附件目录：

```text
attachments/drawings/
```

然后在当前笔记中插入相对路径：

```markdown
![手写绘图标注](attachments/drawings/drawing-2026-06-03T10-30-00-000Z.png)
```

优点：

- Markdown 文件保持干净，不会因为 Base64 图片变得很大。
- 附件跟随 Vault 一起同步，适合 iCloud、GitHub 和普通文件复制。
- 后续可以在附件管理面板里集中查看、替换、删除绘图。

说明：

- 如果只在浏览器演示模式运行，没有本地文件系统桥接，绘图会临时以内联图片方式插入。
- Electron 桌面模式会优先保存为附件文件。

## 5. 平板电脑适配说明

当前桌面 MVP 已使用 Pointer Events 处理输入，因此在支持浏览器 Pointer Events 的设备上，鼠标、触控、手写笔可以共用同一套绘图逻辑。

后续移动端和平板端建议继续扩展：

- 手写笔压感。
- 橡皮擦。
- 多颜色画笔。
- 笔迹撤销/重做。
- 图片上直接标注。
- PDF 或截图标注。
- 绘图附件独立保存。

## 6. 推荐工作流

建议使用方式：

1. 用 Markdown 写正文。
2. 用 `[[双向链接]]` 连接笔记。
3. 用 Frontmatter `tags` 给笔记分类。
4. 用图表代码块表达简单数据。
5. 用媒体链接插入图片、音频、视频。
6. 用绘图面板补充手写思路、草图和批注。
